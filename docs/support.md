# Destek

Ödüyorum ile ilgili hata bildirimi, öneri veya destek talebi için bu public
GitHub deposunun **Issues** bölümünü kullanabilirsiniz.

Bir hata bildirirken mümkünse şu bilgileri ekleyin:

- iPhone veya iPad modeli
- iOS sürümü
- Uygulama sürümü
- Sorunu tekrar oluşturma adımları
- Kişisel veya finansal bilgi içermeyen ekran görüntüsü
